# 🚀 SkillSwap Startup

An AI-powered MERN Stack platform that enables users to exchange skills without money. Users can learn, teach, connect, chat in real-time, and schedule video learning sessions with intelligent skill matching.

## 🌐 Live Demo

🔗 https://skill-swap-startup.vercel.app

---

## 📌 Features

- 🔐 Secure User Authentication (JWT)
- 🤖 AI-powered Skill Matching
- 👤 User Profile Management
- 📚 Add & Manage Skills
- 🔍 Search & Filter Skills
- 💬 Real-Time Chat using Socket.io
- 📹 Video Learning Sessions (WebRTC)
- 📩 Notifications
- 📱 Fully Responsive UI
- ⚡ Fast & Modern MERN Architecture

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sonu1511-sss/SkillSwapStartup.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:5173
```
